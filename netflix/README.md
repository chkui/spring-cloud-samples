**spring eureka 的配置众多，结构上分为四个部分** 

1. instance：对单个实例的控制，在客户端进行配置.
2. client:客户端的设置。在客户端进行配置.
3. server:服务端配置，与服务端提供的内容相关。在eureka服务器进行配置.
4. dashboard:可视化仪表盘相关的配置
